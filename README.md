# 这是一个基于vue3的ui组件库

# 开始
```
npm install
```
在项目的main.js文件中:
```
import 'zcmade_ui/style.css';
import 'zcmade_ui/icon-font/iconfont/css';
```
便可在单一的vue文件中进行按需引入了:
```
<template>
  <c-button></c-button>
</template>
<script setup lang="ts">
import { cButton } from 'zcmade_ui'
</script>
```
组件具体使用详情请看[文档](http://112.74.89.251/)
